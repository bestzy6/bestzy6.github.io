CO-STAR框架

CO-STAR提供了一种结构化的方法来创建Prompt。

# CONTEXT
提供背景信息有助于LLM理解具体场景。

# OBJECT 
明确定义任务可以指导LLM的重点。

# STYLE
指定所需的写作风格将对齐LLM响应。

# TONE
设定语气可确保反应与所需的情绪产生共鸣。

# Audience
定目标受众可以定制LLM的回应，以针对特定受众。

# Response
提供响应格式，如文本或json，确保LLM输出，并帮助构建管道。
